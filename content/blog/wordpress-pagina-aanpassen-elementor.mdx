Elementor is de meest populaire page builder voor WordPress. Met de drag-and-drop editor kun je pagina's visueel ontwerpen zonder code te schrijven. In deze handleiding leer je de basics.

## Wat is Elementor?

Elementor is een page builder plugin voor WordPress die je in staat stelt om:

- Pagina's visueel te ontwerpen
- Elementen te slepen en neer te zetten
- Direct het resultaat te zien
- Responsief te ontwerpen voor mobiel

Er zijn twee versies:
- **Elementor Free**: Basisfeatures, gratis
- **Elementor Pro**: Uitgebreide features, betaald

## Elementor openen

### Bestaande pagina bewerken

1. Ga naar Pagina's → Alle pagina's
2. Hover over de pagina
3. Klik "Bewerken met Elementor"

Of:

1. Ga naar de pagina op je website (uitgelogd)
2. Klik in de admin bar op "Bewerken met Elementor"

### Nieuwe pagina met Elementor

1. Ga naar Pagina's → Nieuwe toevoegen
2. Klik "Bewerken met Elementor"

## De Elementor interface

### Het werkgebied

Het midden van je scherm is het canvas waar je pagina zichtbaar is. Hier sleep je elementen naartoe en zie je direct het resultaat.

### Het linkerpaneel

Hier vind je:

- **Widgets**: Alle elementen die je kunt toevoegen
- **Instellingen**: Opties voor geselecteerde elementen
- **Navigator**: Overzicht van je pagina structuur

### De onderste toolbar

- **Responsive mode**: Switch tussen desktop/tablet/mobiel
- **History**: Bekijk en herstel eerdere versies
- **Preview**: Bekijk de pagina zonder editor

## Basisstructuur: Secties en kolommen

Elementor werkt met een hiërarchie:

```
Sectie (Section)
└── Kolom (Column)
    └── Widget
```

### Sectie toevoegen

1. Klik op het rode + icoon
2. Kies de kolomstructuur (1 kolom, 2 kolommen, etc.)

### Kolommen aanpassen

1. Klik op de kolom handle (grijze lijn bovenaan kolom)
2. Sleep de randen om breedte aan te passen
3. Of stel exacte percentages in via de kolominstellingen

### Sectie-instellingen

Klik op het 6-punts icoon van een sectie voor:

- **Layout**: Breedte, hoogte, gap
- **Achtergrond**: Kleur, afbeelding, video, gradient
- **Border**: Randen en schaduw
- **Responsive**: Verbergen op bepaalde apparaten

## Widgets toevoegen

### Widget naar pagina slepen

1. Zoek de widget in het linkerpaneel
2. Sleep naar de gewenste positie in een kolom
3. Laat los wanneer de blauwe lijn verschijnt

### Veelgebruikte widgets

**Kop (Heading)**
- Voeg koppen toe (H1-H6)
- Pas grootte, kleur en uitlijning aan

**Teksteditor**
- Voor langere tekst met formatting
- Vergelijkbaar met Word

**Afbeelding**
- Upload of selecteer afbeelding
- Pas grootte en uitlijning aan
- Voeg link toe

**Knop (Button)**
- Tekst, link en stijl instellen
- Hover-effecten configureren

**Spacer**
- Voeg witruimte toe tussen elementen
- Stel hoogte in pixels in

**Verdeler (Divider)**
- Horizontale lijn
- Verschillende stijlen beschikbaar

**Icon Box**
- Icoon met tekst
- Handig voor features of diensten

## Elementen bewerken

### Widget selecteren

Klik op de widget om deze te selecteren. Het blauwe kader en de edit-handle verschijnen.

### De drie tabbladen

Elke widget heeft drie tabbladen:

**Content (Inhoud)**
- De daadwerkelijke content
- Tekst, afbeeldingen, links

**Style (Stijl)**
- Visuele opmaak
- Kleuren, fonts, spacing

**Advanced (Geavanceerd)**
- Marges en padding
- Animaties
- CSS classes
- Responsive opties

## Tekst bewerken

### Direct bewerken

1. Dubbelklik op tekst in het werkgebied
2. Bewerk direct inline
3. Klik erbuiten om op te slaan

### Via het paneel

1. Selecteer de widget
2. Bewerk in het Content tabblad
3. Gebruik de rich text editor

### Tekstopmaak

In de teksteditor kun je:
- Bold, italic, underline
- Lijsten maken
- Links toevoegen
- Kleur aanpassen

## Afbeeldingen aanpassen

### Afbeelding vervangen

1. Selecteer de Image widget
2. Klik op de afbeelding in het Content tabblad
3. Upload nieuw of kies uit mediabibliotheek

### Afbeelding instellen

**Image Size**: Kies een voorgedefinieerde grootte
**Alignment**: Links, midden, rechts
**Link**: Maak de afbeelding klikbaar
**Caption**: Voeg onderschrift toe

### Lightbox

1. Bij Link kies je "Media File"
2. Activeer Lightbox (Pro feature of met free plugin)
3. Afbeelding opent groot bij klik

## Knoppen configureren

### Knoptekst en link

1. Selecteer de Button widget
2. In Content: voer tekst en URL in
3. Kies of link in nieuw tabblad opent

### Knopstijl

In het Style tabblad:

- **Typography**: Font, grootte, weight
- **Text Color**: Kleur van de tekst
- **Background**: Achtergrondkleur
- **Border**: Rand en afronding
- **Padding**: Binnenruimte

### Hover state

1. Klik op "Hover" in het Style tabblad
2. Stel afwijkende kleuren in
3. Voeg optioneel animatie toe

## Responsive design

### Preview modes

Gebruik de icoontjes onderin om te schakelen:
- Desktop
- Tablet
- Mobiel

### Per apparaat aanpassen

Veel instellingen hebben een apparaat-icoontje ernaast. Dit betekent dat je per apparaat kunt instellen:

- Font grootte
- Padding/margins
- Kolom breedte
- Verbergen/tonen

### Element verbergen

Wil je iets alleen op desktop tonen?

1. Selecteer het element
2. Ga naar Advanced → Responsive
3. Vink "Hide on Tablet" en "Hide on Mobile" aan

## Kleuren en fonts

### Global kleuren

1. Klik linksboven op het hamburgermenu
2. Ga naar Site Settings
3. Klik Global Colors
4. Stel je merkkeuren in

Nu kun je deze kleuren overal gebruiken via "Global".

### Global fonts

1. Site Settings → Global Fonts
2. Stel Primary, Secondary, Text en Accent fonts in
3. Pas deze toe via "Global" in typography settings

### Kleur kiezen

Bij elke kleuroptie:
1. Klik op het kleurvak
2. Kies uit kleurenpalet
3. Of voer een hex code in (#FF5733)
4. Of kies een Global Color

## Secties opmaken

### Achtergrond instellen

1. Selecteer de sectie (6-punts icoon)
2. Ga naar Style → Background
3. Kies type: Classic, Gradient, Video, Slideshow

**Classic achtergrond:**
- Color: Egale kleur
- Image: Achtergrondafbeelding
- Position: Positie van de afbeelding
- Size: Cover, Contain, Auto

### Overlay

Voeg een kleur of gradient over een achtergrondafbeelding:

1. Style → Background Overlay
2. Kies kleur of gradient
3. Stel opacity in

### Padding en margin

**Padding**: Ruimte BINNEN de sectie
**Margin**: Ruimte BUITEN de sectie

In Advanced → Layout:
- Voer waarden in per zijde
- Of klik link-icoon voor gelijke waarden

## Opslaan en publiceren

### Concept opslaan

Klik onderin op het pijltje naast "Update/Publiceren":
- Save Draft: Opslaan als concept

### Pagina bijwerken

Klik "Update" om wijzigingen live te zetten.

### Pagina publiceren

Nieuwe pagina? Klik "Publish" om live te gaan.

## Templates gebruiken

### Template library

1. Klik op de map-icoon (in een lege sectie)
2. Blader door templates
3. Klik op een template om preview te zien
4. Klik "Insert" om toe te voegen

### Eigen template opslaan

Hergebruik je eigen designs:

1. Klik rechtsboven op het pijltje naast Update
2. Kies "Save as Template"
3. Geef een naam
4. Vind terug in Templates → Saved Templates

## Revisies en herstel

### History gebruiken

1. Klik onderin op het klok-icoon
2. Zie alle wijzigingen
3. Klik op een punt om te herstellen

### Revisions

1. Klik op Revisions (boven Actions)
2. Vergelijk versies
3. Klik "Apply" om een oudere versie te herstellen

## Veelvoorkomende problemen

### Wijzigingen niet zichtbaar

1. Clear je browser cache (Ctrl+Shift+Delete)
2. Leeg eventuele website cache (caching plugin)
3. Check of je wel "Update" hebt geklikt

### Layout verspringt

1. Check responsive instellingen
2. Controleer padding/margin waarden
3. Bekijk in Navigator of structuur klopt

### Kan niet bewerken

1. Zorg dat je bent ingelogd
2. Check of je rechten hebt om te bewerken
3. Schakel conflicterende plugins uit

### Langzame editor

1. Schakel onnodige widgets uit
2. Optimaliseer afbeeldingen
3. Gebruik minder sections

## Tips voor betere pagina's

1. **Consistentie**: Gebruik global kleuren en fonts
2. **Witruimte**: Geef elementen ruimte om te ademen
3. **Hiërarchie**: Maak belangrijke dingen groter/opvallender
4. **Mobile first**: Test altijd op mobiel
5. **Snelheid**: Gebruik geen onnodige effecten
6. **Structuur**: Houd sections logisch georganiseerd

## Elementor vs Gutenberg

Wanneer wat gebruiken?

**Gutenberg (standaard WordPress):**
- Blogposts
- Eenvoudige pagina's
- Content-gerichte pagina's

**Elementor:**
- Landingspagina's
- Homepages
- Portfolio pagina's
- Pagina's met complex design
