On-page SEO omvat alle optimalisaties die je direct op je website kunt doen. Het is wat je volledig onder controle hebt. In deze guide behandelen we alle aspecten van on-page optimalisatie.

## Wat is On-Page SEO?

On-page SEO zijn alle optimalisaties op je eigen pagina's:

- Content optimalisatie
- HTML elementen (titles, meta tags)
- Interne linking
- URL structuur
- Afbeeldingsoptimalisatie

**Verschil met off-page SEO:** Off-page gaat over externe factoren zoals backlinks en social signals.

## Title Tags

De title tag is het belangrijkste on-page SEO element.

### Waar Verschijnt de Title?

- In het browser tabblad
- In zoekresultaten als klikbare kop
- Bij social media sharing

### Best Practices

**Lengte:** 50-60 karakters (Google knipt af bij ~60)

**Structuur:**
```
Primary Keyword - Secondary Keyword | Brand
```

**Voorbeelden:**
```
✓ Webdesign Den Haag | Professionele Websites | Robuust
✓ SEO Tips voor Beginners: Complete Gids [2025]
✗ Home | Robuust Marketing
✗ Welkom op onze website over webdesign en development
```

### Checklist Title Tags

- [ ] Primary keyword vooraan
- [ ] Uniek per pagina
- [ ] Maximaal 60 karakters
- [ ] Aantrekkelijk om op te klikken
- [ ] Brand naam opgenomen

## Meta Descriptions

De meta description is je "advertentie" in zoekresultaten.

### Waar Verschijnt Het?

Onder de title in zoekresultaten. Google kan deze overschrijven met eigen tekst.

### Best Practices

**Lengte:** 150-160 karakters

**Elementen:**
- Relevante keywords (worden vet weergegeven)
- Call-to-action
- Unique selling point
- Urgentie of voordeel

**Voorbeelden:**
```
✓ Professioneel webdesign in Den Haag. Snelle, SEO-vriendelijke websites
  voor MKB. Gratis adviesgesprek. Bekijk onze portfolio!

✗ Robuust Marketing is een webdesign bureau gevestigd in Den Haag dat
  websites maakt voor bedrijven.
```

### Checklist Meta Descriptions

- [ ] Inclusief target keyword
- [ ] Call-to-action aanwezig
- [ ] Uniek per pagina
- [ ] Maximaal 160 karakters
- [ ] Compelling en clickworthy

## Heading Structuur (H1-H6)

Headings geven structuur aan je content en helpen zoekmachines begrijpen waar de pagina over gaat.

### H1 Tag

**Regels:**
- Slechts één H1 per pagina
- Bevat primary keyword
- Beschrijft de hoofdinhoud
- Uniek per pagina

**Voorbeeld:**

`Webdesign Den Haag: Professionele Websites voor Jouw Bedrijf`

### H2-H6 Hiërarchie

Gebruik headings als outline:

```
H1: Webdesign Den Haag
├── H2: Onze Webdesign Diensten
│   ├── H3: Zakelijke Websites
│   ├── H3: Webshops
│   └── H3: WordPress Development
├── H2: Waarom Kiezen voor Ons?
│   ├── H3: Ervaring
│   └── H3: Resultaten
└── H2: Neem Contact Op
```

### Heading Tips

- Gebruik keywords in H2's
- Maak headings scannable
- Sla geen levels over (H1 → H3 is fout)
- Niet te veel H2's (5-8 is goed)

## Content Optimalisatie

### Content Lengte

**Geen vaste regel, maar:**
- Thin content (&lt;300 woorden) rankt zelden goed
- Diepgaande content (1500+ woorden) rankt vaak beter
- Kwaliteit > Kwantiteit

### Keyword Plaatsing

**Waar keywords opnemen:**
- Title tag
- H1 heading
- Eerste 100 woorden
- H2 headings (sommige)
- Afbeelding alt tags
- URL
- Meta description

**Keyword density:** Er is geen ideaal percentage. Schrijf natuurlijk.

### LSI Keywords

Latent Semantic Indexing keywords zijn gerelateerde termen:

**Primary keyword:** webdesign
**LSI keywords:** website ontwerp, webdeveloper, responsive design, UX design

Google begrijpt context door gerelateerde termen.

### Content Structuur

**Voor leesbaarheid:**
- Korte paragrafen (2-3 zinnen)
- Bullet points en lijsten
- Subheadings om content te breken
- Visuele elementen

**Voor SEO:**
- Beantwoord de zoekvraag direct
- Belangrijke info bovenaan
- Duidelijke conclusie

### E-E-A-T Signalen

Google waardeert Experience, Expertise, Authoritativeness, Trust:

**Experience:**
- Deel eigen ervaringen
- Case studies
- Praktische voorbeelden

**Expertise:**
- Auteur bio's
- Credentials vermelden
- Diepgaande kennis tonen

**Authoritativeness:**
- Citeer bronnen
- Link naar gezaghebbende sites
- Verdien backlinks

**Trust:**
- Over ons pagina
- Contactgegevens
- Privacy policy
- Reviews/testimonials

## URL Optimalisatie

### URL Best Practices

**Goed:**
```
/diensten/webdesign
/blog/seo-tips-beginners
/portfolio/website-bedrijf-x
```

**Slecht:**
```
/page.php?id=123
/diensten-en-producten-die-wij-aanbieden/webdesign-services
/Blog/SEO-Tips
```

### URL Checklist

- [ ] Kort en beschrijvend
- [ ] Keyword opgenomen
- [ ] Lowercase
- [ ] Hyphens als scheidingsteken
- [ ] Geen stop words (de, het, een)
- [ ] Geen speciale tekens

## Afbeeldingsoptimalisatie

### Alt Tekst

Alt tekst beschrijft afbeeldingen voor:
- Zoekmachines
- Screen readers (toegankelijkheid)
- Wanneer afbeelding niet laadt

**Voorbeelden:**

- ✓ alt="Modern webdesign voor restaurant website"
- ✓ alt="Team van Robuust Marketing aan het werk"
- ✗ alt="image1.jpg"
- ✗ alt="webdesign webdesign webdesign den haag"
- ✗ alt="" (leeg)

### Bestandsnamen

**Goed:**
```
webdesign-portfolio-restaurant.webp
team-robuust-marketing.jpg
```

**Slecht:**
```
IMG_1234.jpg
screenshot-2024-01-15.png
```

### Afbeelding Compressie

- WebP formaat gebruiken
- Lazy loading implementeren
- Juiste afmetingen serveren
- Quality vs filesize balanceren

### Image Checklist

- [ ] Beschrijvende alt tekst
- [ ] Keyword in alt waar relevant
- [ ] Beschrijvende bestandsnaam
- [ ] Gecomprimeerd formaat
- [ ] Width en height attributen

## Interne Linking

### Waarom Belangrijk?

- Helpt zoekmachines structuur begrijpen
- Verspreidt "link juice"
- Verbetert gebruikerservaring
- Verlaagt bounce rate

### Interne Linking Strategie

**1. Link naar belangrijke pagina's**
Je homepage en service pagina's moeten veel interne links krijgen.

**2. Gebruik beschrijvende anchor text**

- ✓ Lees meer over onze **webdesign diensten** (beschrijvende anchor)
- ✗ **Klik hier** voor meer info (niet beschrijvend)

**3. Link vanuit content**
Niet alleen navigatie, maar ook binnen de tekst.

**4. Gerelateerde content linken**
Blog posts linken naar relevante andere posts.

### Checklist Interne Links

- [ ] Alle pagina's hebben minstens 1 interne link
- [ ] Belangrijke pagina's hebben meerdere links
- [ ] Anchor text is beschrijvend
- [ ] Links zijn relevant voor de context
- [ ] Geen broken internal links

## Schema Markup

Structured data helpt zoekmachines content beter te begrijpen.

### Belangrijke Schema Types

**Article (voor blog posts):**
```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "On-Page SEO Guide",
  "author": {
    "@type": "Organization",
    "name": "Robuust Marketing"
  },
  "datePublished": "2024-01-15"
}
```

**FAQ:**
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "Wat is on-page SEO?",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "On-page SEO omvat alle optimalisaties..."
    }
  }]
}
```

**LocalBusiness (voor lokale bedrijven):**
Naam, adres, openingstijden, reviews.

### Schema Testen

- Google Rich Results Test
- Schema Markup Validator

## Page Experience Signals

### Core Web Vitals

- **LCP:** Largest Contentful Paint &lt; 2.5s
- **INP:** Interaction to Next Paint &lt; 200ms
- **CLS:** Cumulative Layout Shift &lt; 0.1

### Mobile Friendliness

- Responsive design
- Leesbare tekst zonder zoomen
- Voldoende ruimte tussen tap targets

### HTTPS

SSL certificaat is verplicht.

### No Intrusive Interstitials

Vermijd:
- Full-screen popups
- Ads die content verbergen
- Verplichte app-install banners

## On-Page SEO Audit Checklist

### Per Pagina Controleren

**Title & Meta:**
- [ ] Unieke, geoptimaliseerde title tag
- [ ] Unieke meta description
- [ ] Correcte canonical tag

**Content:**
- [ ] Eén H1 met keyword
- [ ] Logische heading hiërarchie
- [ ] Keyword in eerste 100 woorden
- [ ] Voldoende content lengte
- [ ] Interne links aanwezig

**Technisch:**
- [ ] Korte, beschrijvende URL
- [ ] Afbeeldingen geoptimaliseerd
- [ ] Schema markup geïmplementeerd
- [ ] Pagina laadt snel
- [ ] Mobile friendly

**Gebruikerservaring:**
- [ ] Scannable content
- [ ] Visuele elementen
- [ ] Duidelijke CTA
- [ ] Geen intrusive elements

## Veelgemaakte Fouten

### 1. Keyword Stuffing

**Fout:** Hetzelfde keyword 50 keer op een pagina.
**Fix:** Schrijf natuurlijk, gebruik variaties.

### 2. Duplicate Title Tags

**Fout:** Alle pagina's dezelfde title.
**Fix:** Unieke, beschrijvende titles per pagina.

### 3. Dunne Content

**Fout:** Pagina's met 100 woorden tekst.
**Fix:** Voeg waarde toe of consolideer pagina's.

### 4. Vergeten Alt Tekst

**Fout:** Afbeeldingen zonder alt attributen.
**Fix:** Beschrijf elke afbeelding.

### 5. Slechte Interne Linking

**Fout:** Alleen navigatie links.
**Fix:** Link ook vanuit content naar relevante pagina's.

On-page SEO is wat je volledig in eigen hand hebt. Focus op:

1. **Fundamentals:** Title, meta, headings
2. **Content:** Kwaliteit, structuur, relevantie
3. **Technisch:** Snelheid, mobile, schema
4. **Links:** Interne linking strategie

Begin met een audit van je belangrijkste pagina's en verbeter systematisch.
